# Narwhals Data Reader

## How to use it

#### Without commands
```shell
java -jar target/rn-data-reader.jar
```

#### Ask for help specifically

```shell
java -jar target/rn-data-reader.jar
```

#### Example with a file

```shell
java -jar target/rn-data-reader.jar -f ../commons/src/test/resources/narwhals1.xml -d 13
```
